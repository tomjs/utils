> 纯 `JavaScript` 库，可以运行在 `浏览器` 和 `node.js`

## 支持环境

- Node.js >=16
- Chrome >=87
- Firefox >=78
- Safari >=14
- Edge >=88

## 安装

```bash
# pnpm
pnpm add @tomjs/utils

# yarn
yarn add @tomjs/utils

# npm
npm i @tomjs/utils
```
