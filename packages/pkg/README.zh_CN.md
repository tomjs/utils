> npm 包工具

## 特性

- 获取包管理器类型（`npm`, `pnpm`, `yarn`, `berry`）和版本号

## 安装

```bash
# pnpm
pnpm add @tomjs/pkg -D

# yarn
yarn add @tomjs/pkg -D

# npm
npm add @tomjs/pkg -D
```
