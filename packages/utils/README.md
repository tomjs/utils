## Support environment

- Node.js >=16
- Chrome >=87
- Firefox >=78
- Safari >=14
- Edge >=88

## Install

```bash
# pnpm
pnpm add @tomjs/utils

# yarn
yarn add @tomjs/utils

# npm
npm i @tomjs/utils
```
